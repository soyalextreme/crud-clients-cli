CRUD FOR MANAGING CLIENTS

To set up this CLI you need to follow this steps:

1- create a directory
2- Clone this repository
3- Create a virtual env with python
      python3 -m venv env
4- With venv activate install click
      pip install click
5- Install the cli
      pip install .
      pip install --editable .
(add editable flag if you wanna update or modify the code)
